Take credit card payments on your store using Payselection.

== Description ==

Accept Visa, MasterCard, Maestro, Мир on your store with the Payselection payment gateway for WooCommerce.

= Take Credit card payments easily and directly on your store =

The Payselection plugin extends WooCommerce allowing you to take payments directly on your store via Payselection’s API.

== Installation ==

1. Log in to WordPress admin.
2. Go to **Plugins > Add New**.
3. Search for the **Payselection Gateway for WooCommerce** plugin.
4. Click on **Install Now** and wait until the plugin is installed successfully.
5. You can activate the plugin immediately by clicking on **Activate** now on the success page. If you want to activate it later, you can do so via **Plugins > Installed Plugins**.

== Frequently Asked Questions ==

= Does this require an Payselection merchant account? =

Yes! An Payselection merchant account is required for this gateway to function.
